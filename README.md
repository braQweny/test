# Validate-Types
simple function to validate types of variables
### Example
`VALIDATE.toType(123) ==> "number"`
### Run Test
```sh
$ npm install jasmine-node -g
$ cd "absolute path of project's folder"
$ jasmine-node test/
```
**OR** 
``` sh
$ cd "absolute path of project's folder"
$ npm install 
$ node node_modules/jasmine-node/lib/jasmine-node/cli.js test/
```
